Game rules

1. At the start, Simon will randomly give you a signal (one color will pop up).
2. You have to repeat the signal by pressing the same color.
3. Simon will add a new random signal.
4. You need to repeat both the previous and the new signal by tapping color lenses in the order.
5. Simon will keep on adding new random signals.
6. Continue playing as long as you can repeat the sequence.
